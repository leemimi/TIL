## 5.생성자
### 5.1 생성자란?
인스턴스 초기화 메서드

메서드 클래스 내에 선언, 리턴값이 없다

1. 생성자의 이름은 클래스의 이름과 같다
2. 생성자는 리턴값이 없다 -> void생략

```
클래스이름 (타입 변수명, 타입변수명, ..) {
    //인스턴스 생성 시 수행 될 코드
    // 주로 인스턴스 변수의 초기화 코드
}
```

Card c = new Card();
1. 연산자 new에 의해서 메모리(heap)에 Card클래스의 인스턴스가 생성
2. 생성자 Card()가 호출되어 수행됨
3. 연산자 new의 결과로, 생성된 Card의 인스턴스 주소가 반환되어 참조변수 c에 저장된다.
